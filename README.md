# Модульна контрольна робота з ООП

## НД “Об'єктно орієнтоване програмування”

### МКР Рівень 1

У репозиторії розміщено практичні C++ приклади до контрольних завдань першого рівня модульної контрольної роботи.

Кожне завдання оформлено в окремій папці. Для кожного прикладу додано `.cpp` файл з кодом та скріншот результату виконання програми в терміналі.

## Структура роботи

| № | Тема | Файл |
|---|---|---|
| 1 | Загальні форми `try`, `catch`, `throw` | `1/01_try_catch_throw.cpp` |
| 2 | Сумісна робота `try`, `catch`, `throw` | `2/02_exception_handling_example.cpp` |
| 3 | Використання `throw` без `try` | `3/03_throw_without_try.cpp` |
| 4 | Переваги введення-виведення C++ порівняно з C | `4/04_cpp_io_advantages.cpp` |
| 5 | Функції та маніпулятори форматування | `5/05_format_parameters.cpp` |
| 6 | Factory Method | `6/06_factory_method.cpp` |
| 7 | Dependency Injection | `7/07_dependency_injection.cpp` |
| 8 | Visitor | `8/08_visitor_pattern.cpp` |

## Компіляція

Для компіляції використовувався Microsoft C++ Compiler у середовищі Developer PowerShell for Visual Studio.

Приклад компіляції:

```powershell
cl /EHsc filename.cpp
